Análisis Exploratorio de Datos sobre Fumadores extraido de kaggel.

Descripción del Proyecto:

Este proyecto tiene como objetivo realizar un análisis detallado de datos relacionados con el tabaquismo. Se exploran diversos aspectos del comportamiento de los fumadores,
así como su relación con variables demográficas y de estilo de vida.

Contenido:

Datos: Se utilizan conjuntos de datos que contienen información sobre fumadores y no fumadores, así como datos demográficos y de hábitos de vida.
Notebooks de Jupyter: Se incluyen varios notebooks de Jupyter que detallan el proceso de análisis y visualización de datos.
Visualizaciones: Se presentan visualizaciones interactivas y estáticas que ayudan a comprender mejor los patrones y tendencias relacionadas con el tabaquismo.
Conclusiones: Se resumen las principales conclusiones y hallazgos obtenidos durante el análisis exploratorio.
Objetivos
Analizar la distribución de fumadores y no fumadores en la muestra de datos.
Explorar la relación entre el hábito de fumar y variables demográficas como la edad, el género y el nivel educativo.
Investigar patrones de consumo de tabaco, incluyendo la cantidad de cigarrillos fumados por día y la duración del hábito.
Identificar factores de riesgo asociados al tabaquismo, como el nivel socioeconómico y el estado de salud.
Proponer recomendaciones basadas en los hallazgos para programas de prevención del tabaquismo y políticas de salud pública.
Tecnologías Utilizadas
Python
Jupyter Notebook
Pandas
Matplotlib
Seaborn
Power BI (para visualizaciones interactivas)

Resultados : 
Se observa una prevalencia significativa de fumadores en la muestra de datos, con una distribución desigual entre diferentes grupos demográficos.
La edad y el género son factores importantes que influyen en el hábito de fumar, con una tendencia a un mayor consumo entre hombres y personas de mayor edad.
Se identifican patrones de consumo variados, incluyendo diferencias en la cantidad de cigarrillos fumados por día y la duración del hábito.
Factores socioeconómicos, como el nivel educativo y el ingreso, muestran asociaciones con el hábito de fumar, destacando la necesidad de enfoques diferenciados en las políticas de prevención.
